## 🚀 Utilisation

### Fichiers de Cours

Chaque dossier contient:
- `*_init.html` : Fichiers de départ pour les exercices
- `*_2025.html` : Fichiers de travail pour l'année en cours
- `*_solution.html` : Solutions complètes
- `readme.txt` : Instructions spécifiques

### Exemples Pratiques (Nouveaux)

Les nouveaux exemples sont autonomes (HTML, CSS et JS dans un seul fichier):
1. Ouvrez le fichier HTML dans votre navigateur
2. Appuyez sur F12 pour voir la console
3. Lisez les commentaires dans le code
4. Modifiez et expérimentez!
