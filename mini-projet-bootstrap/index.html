Qu'est-ce que Bootstrap?

Bootstrap est le framework CSS le plus populaire au monde pour créer des sites web responsives et mobile-first.

  • Responsive - S'adapte à tous les écrans
  • Composants - Plus de 50 composants prêts à l'emploi
  • Grid System - Système de grille flexible
  • Utilities - Classes utilitaires pour un styling rapide
